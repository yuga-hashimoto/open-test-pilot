# Support

Open an issue at [github.com/yuga-hashimoto/open-test-pilot/issues](https://github.com/yuga-hashimoto/open-test-pilot/issues) for reproducible bugs and feature requests.

For usage questions, include the OpenTestPilot version, Node.js and pnpm
versions, the runner mode, and a redacted diagnostic log. Never attach secrets
or customer artifacts. Feature requests should describe the workflow and
acceptance criteria rather than only a preferred implementation.
