# Repository map

This is the compact orientation page for people and coding agents. Paths below are source paths unless marked generated.

| Area | Entry point | Owns | Smallest useful check |
| --- | --- | --- | --- |
| Manifest contract | `packages/manifest-schema/src` | schema, IDs, actions, safety constraints | `pnpm --filter @open-test-pilot/manifest-schema test` |
| Manifest parsing | `packages/manifest-parser/src` | YAML parsing and diagnostics | `pnpm --filter @open-test-pilot/manifest-parser test` |
| Migration | `packages/manifest-migrator/src` | version migration and diff | `pnpm --filter @open-test-pilot/manifest-migrator test` |
| Generation | `packages/generator/src` | Playwright/Appium source generation | `pnpm --filter @open-test-pilot/generator test` |
| Local execution | `packages/local-runner/src` | isolated runs and reports | `pnpm --filter @open-test-pilot/local-runner test` |
| CLI | `packages/cli/src` | `testpilot` commands | `pnpm --filter @open-test-pilot/cli test` |
| Control plane | `apps/server/src` | API, auth, tenancy, persistence | `pnpm --filter @open-test-pilot/server test` |
| Dashboard | `apps/web/src` | human-facing UI | `pnpm test:web:ui` |
| Runner/scheduler | `apps/runner`, `apps/scheduler` | distributed execution and schedules | package test + local Compose |
| AI worker | `apps/ai-worker` | policy-gated analysis/repair | `pnpm --filter @open-test-pilot/ai-worker test` |
| Protocols | `packages/agent-protocol`, `runner-protocol`, `result-schema` | stable JSON contracts | corresponding package test |
| Integrations | `packages/*-adapter` | GitHub, API, storage, queue, secrets, mobile | corresponding package test |

## Generated and ignored paths

- `dist/` and `.testpilot/` are build/run output; never use them as source.
- `examples/manifests/generated/` is generated from the example manifests. Use `pnpm verify:generated` after changing a source manifest or generator.
- `docs/THIRD_PARTY_LICENSES.json` is generated by `pnpm license:report`.

## Verification tiers

- Fast: focused package test and `pnpm typecheck`.
- Repository: `pnpm test`, `pnpm build`, `pnpm verify:generated`.
- Browser: `pnpm test:web:ui` (starts the Vite dashboard automatically).
- Operator/external: Docker/Helm, real devices, registry publication, hosted deployment, and credentials are separate gates; see `docs/CAPABILITY_STATUS.md`.
