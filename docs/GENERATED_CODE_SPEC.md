# Generated Code Specification

Generation is deterministic by normalized Manifest and generator version. Web output is standard Playwright TypeScript; mobile output is WebdriverIO/Appium TypeScript; API output uses standard TypeScript HTTP APIs. Each generated test includes stable node annotations, a thin optional reporter fixture, a source map, and a standalone README.

Generated code is an artifact by default, can be committed under `generated/` by project policy, and can be exported as an independent project or ZIP. The generator must fail on unsupported semantics instead of producing code that silently omits a node.
